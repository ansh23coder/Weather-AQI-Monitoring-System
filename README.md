# Smart Weather App

A simple Python application that shows the current weather and air quality index (AQI) for a given city using Tkinter for the GUI and pyttsx3 for text-to-speech functionality.

## Features
- Displays temperature, humidity, wind speed, and weather condition.
- Provides air quality information (PM2.5, PM10, AQI).
- Uses text-to-speech to read out weather details and provide warnings for unhealthy air quality.

- Python 3.x
